# 🤖 Sahabat Pintar Anak – AI Chatbot

Chatbot AI ramah anak untuk edukasi. Menyediakan jawaban sesuai umur: 👶 Anak Kecil (4–8 tahun) dengan jawaban singkat penuh emoji, 🧑 Anak Besar (9–12 tahun) dengan jawaban lebih detail. Fitur: filter konten aman, riwayat percakapan, reset chat, dark mode permanen.

---

## 📌 Use Case
Education Bot untuk anak-anak:
- Membantu anak belajar dengan cara menyenangkan.
- Jawaban sesuai umur (anak kecil vs anak besar).
- Filter konten agar percakapan tetap aman.

---

## ✨ Fitur Utama
- Riwayat percakapan.
- Reset chat.
- Filter konten.
- Mode personal (anak kecil / anak besar).
- Dark mode permanen.

---

## 🛠️ Teknologi
- Frontend: HTML, CSS, JavaScript.
- Backend: Node.js + Express.
- AI Model: Google Generative AI (Gemini).
